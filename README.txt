execute ControlMeQuick in a terminal. 
The Program requires at least one argument, up to two.

argument 1: Level name, eg. "CoolLevel.dat"
argument 2: OPTIONAL. Image sequence directory relative to the executable. 
	    Please add a '/' at the end of the filepath. (eg. My/File/Path/)

examples of execution:
./AnimateMeQuick MyLevel.dat
./AnimateMeQuick MyLevel.dat My/Folders/

FOR THIS EXAMPLE CONFIGURATION, THE EXECUTION LINE IS:
./AnimateMeQuick Level1.dat
